Fake News Detection Using Machine Learning

This project aimed to develop a machine learning model to detect and classify news articles as fake or true. By using algorithms such as Logistic Regression, Random Forest, Gradient Boosting, and Decision Trees, the project sought to address the growing issue of misinformation by automatically identifying false news articles.

  • Technologies Used: Python, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn, TF-IDF
  • Challenges Faced: Data imbalance, Model optimization, Feature selection
  • Outcomes: High accuracy with Logistic Regression (90%), Random Forest (88%)
  • Model Evaluation: Precision, recall, F1 score, and accuracy were used to evaluate the models' performance.
  • Real-world Impact: Potential for integration into news platforms and social media sites for automatic fake news detection.

StyleSync: Fashion Recommender App

StyleSync is a mobile application designed to streamline wardrobe management and outfit planning while promoting social engagement. The app offers an intuitive platform where users can visualize how clothing items will look on their body shape through a virtual mannequin.

  • Technologies Used: Swift, SwiftUI, RealityKit, SceneKit, Blender, GitHub, CocoaPods/Swift Package Manager
  • Challenges Faced: 3D Rendering, User interface design, Customization features
  • Outcomes: Successful prototype developed with interactive virtual wardrobe and future features planned (AI-driven outfit suggestions, AR try-ons)

Arduino OLED Stopwatch

The Arduino OLED Stopwatch is a simple yet effective Arduino-based system that features an OLED display for showing time and a push button for switching between different modes.

  • Technologies Used: Arduino NANO, OLED SSD1306, Adafruit GFX, C++
  • Challenges Faced: Difficulty in controlling button presses for switching modes, Time display resetting issues
  • Outcomes: Fully functional stopwatch with mode switching and hands-on experience with hardware interfacing

Dice Roller Simulator

The Dice Roller Simulator simulates the rolling of one or two dice, with the result displayed on an OLED screen.

  • Technologies Used: Arduino UNO/NANO, OLED SSD1306, Adafruit GFX, C++
  • Challenges Faced: Managing multiple dice rolls, Button press debounce issues
  • Outcomes: Interactive dice rolling with dynamic display updates and buzzer for feedback
